# pharmaTools

-findDrugs, this code receives a genes.csv file with ENSEMBL_GENE_ID gene format and finds drug information connected to that gene. If the genes are not in ENSEMBL_GENE_ID gene format, use gene id Conversion tool from https://davidbioinformatics.nih.gov/conversion.jsp

-processGSE, this code helps process datasets from the gene expression omnibus https://www.ncbi.nlm.nih.gov/geo/
